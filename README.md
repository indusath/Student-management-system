# Student Management System (SMS) - Microservices Project

This project is a **Student Management System (SMS)** developed using a modern **Microservices Architecture**.  
It focuses on **scalability, maintainability, and portability**, and is fully deployable using Docker containers.

---

## 🚀 Getting Started

To quickly run the project, follow the instructions below. The system is fully containerized, so no manual setup of dependencies is required.

---

## 📘 Running the Project

### ✅ Prerequisites

- Install and run **Docker Desktop**
- No need to install Java, Maven, or any IDE
